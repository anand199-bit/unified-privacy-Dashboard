# UPDA - Unified Privacy Dashboard & App Privacy Manager

🌆 **RETRO NEON CYBERPUNK PRIVACY MANAGER** 🌆

A futuristic Android app built with Jetpack Compose featuring retro 80s aesthetics and neon cyberpunk styling.

## Features

- **Permission Manager**: Real-time app permission tracking with neon toggles
- **AI Risk Analyzer**: ML-based risk scoring with animated gauges
- **Encrypted Logs**: AES-256 encrypted activity logs
- **Parental Control**: App blocking and time limits
- **Compliance Center**: GDPR, CCPA, COPPA compliance status

## Tech Stack

- Jetpack Compose + Material 3
- MVVM Architecture
- Hilt (Dependency Injection)
- Room Database
- Kotlin Coroutines + StateFlow
- AES-256 Encryption

## Build

```bash
# Open in Android Studio Hedgehog or later
# Sync Gradle
# Run on Android 8.0+ (API 26+)
```
